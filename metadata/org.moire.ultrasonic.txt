Description:
UltraSonic is a Subsonic client for Android. You can use UltraSonic to connect
to a Subsonic (or Subsonic-compatible) server and listen to music.

Main features:

* Thin
* Fast
* Dark and light theme
* Multiple server support
* Offline mode
* Bookmarks
* Playlists on server
* Ramdom play
* Jukebox mode
* Server chat
* And much more!!!
