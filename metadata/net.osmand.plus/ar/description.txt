OsmAnd (اOSM توجيهات الملاحة الآلية) تطبيق الخرائط والملاحة مع امكانية الوصول إلى بيانات (OSM) مجاناً وفي جميع أنحاء العالم، وبجودة عالية. استمتع بمستكشف بالصوت والصورة، وعرض POI (نقاط الاهتمام)، إنشاء وإدارة مسارات GPX، استخدام عرض وعلو بيانات الخطوط الكنتورية (عن طريق البرنامج المساعد)، اختيار بين نمط القيادة، ركوب الدراجات وطرق المشاة، التعديل على الموقع وغيرها.

إنّ OSMAnd+ هي النسخة الكاملة لهذا التطبيق. تبرعوا لمساندة المشروع مِن فضلكم و لتمويل تطوير ميزات جديدة و لكي تتحصلوا على آخر التحديثات.

بعض الميزات الرئيسية: \ n ملاحة \ n • يعمل عبر الإنترنت (سريع) أو غير متصل (لا توجد رسوم تجوال عندما تكون بالخارج) \ n • توجيه صوتي مفصّل (أصوات مسجلة ومركبة) \ n • توجيه اختياري للمسار ، عرض اسم الشارع والوقت المقدر للوصول \ n • دعم النقاط الوسيطة في خط سير الرحلة \ n • إعادة التوجيه تلقائيًا كلما انحرفت عن المسار \ n • ابحث عن الأماكن حسب العنوان ، حسب النوع (على سبيل المثال: مطعم ، فندق ، غاز محطة أو متحف) أو حسب الإحداثيات الجغرافية

عرض الخريطة \ n • اعرض موقعك واتجاهك \ n • قم بمحاذاة الصورة بشكل اختياري وفقًا للبوصلة أو اتجاه حركتك \ n • احفظ الأماكن الأكثر أهمية كمفضلة \ n • عرض النقاط المهمة (نقاط الاهتمام) من حولك \ n • عرض مربعات مخصصة عبر الإنترنت وعرض القمر الصناعي (من Bing) وتراكبات مختلفة مثل مسارات GPX للتجول / التنقل وطبقات إضافية بشفافية قابلة للتخصيص \ n • عرض أسماء الأماكن اختياريًا باللغة الإنجليزية أو المحلية أو الصوتية
