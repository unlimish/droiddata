Альтернативная мультиязычная экранная клавиатура

* Поддержка многоязычной клавиатуры с помощью внешних пакетов (в стандартной комплектации - английский)
* Дополнение словарей для нескольких языков
* Поддержка Multitouch
* Расширенная клавиатура (проведите пальцем по всей клавиатуре вверх)
* Поддержка жестов: проведите влево или вправо для переключения раскладки; проведите вверх для переключения; проведите вниз для закрытия клавиатуры
* Поддержка тем (поставляется с несколькими скинами)
* Клавиатура Emoji плюс поиск по тегам emoji - введите ":
* Встроенный редактор слов пользовательского словаря
* Дополнительная 16-клавишная английская клавиатура
* Софт-клавиши перемещения курсора, которые также могут быть сопоставлены с аппаратными клавишами
* Поддержка ARM, x86 и MIPS нативно
* Голосовой ввод
* Возможность переопределения значения системной локали

Мы стараемся публиковать все языковые пакеты и темы, у которых есть исходные xml, установленный автор и лицензия для словарей. С подробностями можно ознакомиться на fdroid.org [http://f-droid.org/forums/topic/anysoftkeyboard-language-packs-layouts-plans/#post-6408 forum topic].
