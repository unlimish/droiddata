دسترسی به پوستهٔ خط فرمان لینوکس توکار اندروید.

نگارش جدید نرم‌افزار پرهوادار «شبیه‌ساز پایانهٔ اندروید». همان برنامه، با نام جدید.

ویژگی‌های برتر

+ شبیه سازی کامل پایانهٔ گنو/لینوکس.
+ چند پنجره‌ای.
+میان‌برهای اجراگر.
+ متن UTF-8 (فارسی، چینی، یونانی، عبری، ژاپنی، کره‌ای، روسی، تایلندی و…)
+ کاملا آزاد. بدون تبلیغات، بدون خریدهای درون‌برنامه‌ای، بدون غر زدن.

پرسش‌های پرتکرار:

+ اگر برای وارد کردن متن مشکلی دارید، سعی کنید Hacker's Keyboard IME را نصب کنید. ( مخصوصا اگر افزارهٔ سامسونگ یا HTC دارید.)
+ این کاره، شبیه‌ساز بازی نیست.
+ این کاره به روت کردن یاتغییر IMEI تلفنتان کمکی نمی‌کند.
+ برای استفاده از این برنامه باید چگونگی استفاده از خط فرمان گنو/لینوکس را بدانید (یا بخواهید یاد بگیرید).
+ شاید بخواهید برای افزودن دستورهایی، بیش از آن‌چه در افزارهٔ اندرویدیتان ساخته شده، مجموعه ای از ابزارهای خط فرمانی مثل Busy Box یا Debian chroot را نصب کنید.
+ اگر هنگام تلاش برای نصب این کاره روی اندروید ۵، خطای ۵۰۵ گرفتید،یعنی کارهٔ دیگری در حال استفاده از همین اجازه‌هاست. باید برای نصب شبیه ساز پایانه، کارهٔ دیگر را حذف نصب کنید. (برخی گفته‌اند jrummy's Toolbox Pro باعث بروز این مشکل می‌شود.)
