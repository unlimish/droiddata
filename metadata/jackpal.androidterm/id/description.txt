Akses shell baris perintah Linux bawaan Android.

Versi baru dari aplikasi "Android Terminal Emulator" yang populer. Program yang sama, nama baru.

Fitur Teratas

+ Emulasi penuh terminal linux.
+ Banyak jendela.
+ Pintasan peluncur.
+ Teks UTF-8. (Arab, Cina, Yunani, Ibrani, Jepang, Korea, Rusia, Thai, dll.)
+ Sepenuhnya gratis. Tanpa iklan, tanpa pembelian dalam aplikasi, tanpa layar yang menggaggu.

FAQ Ringkas:

+ Jika anda menghadapi kesulitan dalam memasukkan kata, coba untuk instalasi perangkat lunak " Hacker's Keyboard IME". (Khususnya jika anda menggunakan perangkat Samsung atau HTC.)
+ Aplikasi ini bukan sebuah emulator untuk permainan.
+ Aplikasi ini tidak dapat digunakan untuk melakukan proses pembongkaran IMEI atau ponsel anda.
+ Anda harus memastikan memahami ( atau berkeinginan untuk mempelajarinya) untuk memastikan bagaimana menggunakan perintah Linux dalam aplikasi ini.
+Anda mungkin ingin menginstal satu set utilitas baris perintah seperti "Kotak Sibuk" atau "chroot Debian" untuk menambahkan perintah di luar perintah yang ada di perangkat Android Anda.
+ dan ketika anda mendapati kasus -505 pada proses instalasi aplikasi ini pada sistem android versi 5 kebawah, itu berarti ada satu atau beberapa aplikasi lainnya yang juga menggunakan perijinan yang sama. Anda harus menghapus instalasi beberapa aplikasi lainnya tadi, jika anda ingin menggunakan Emulator terminal untuk android. (Banyak orang mengatakan ini diakibatkan oleh " kotak peralatannya jrummy .)
