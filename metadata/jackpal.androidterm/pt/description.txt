Aceda à linha de comandos shell Linux integrada do seu Android.

Nova versão da aplicação popular "Android Terminal Emulator". O mesmo programa, apenas com um nome novo.

Funções principais

+ Emulação completa do terminal Linux.
+ Múltiplas janelas.
+ Atalhos para lançadores.
+ Texto UTF-8. (Árabe, Chinês, Grego, Hebraico, Japonês, Coreano, Russo, Tailandês etc.)
+ Inteiramente gratuito. Sem anúncios, sem compras na aplicação, sem ecrãs irritantes, nada.

FAQ rápido:

+ Se estiver a ter problemas ao digitar o texto, experimente instalar a app gratuita "Hacker's Keyboard IME" (especialmente se tiver um aparelho Samsung ou HTC).
+ Esta app não é um emulador de jogos.
+ Esta app não ajuda a fazer root no telefone ou mudar o IMEI.
+ Tem de saber (ou estar disposto a aprender) como usar a linha de comandos do Linux para usar esta app.
+ Talvez queira instalar um conjunto de utilitários de linha de comandos como o "Busy Box" ou "Debian chroot" para adicionar comandos além dos que vêm integrados no seu aparelho Android.
+ Se vir um erro -505 ao tentar instalar esta app no Android 5.0, significa que outra app usa as mesmas permissões. Terá de desinstalar a outra app para poder instalar o Terminal Emulator for Android (dizem que o "jrummy's Toolbox Pro" causa este problema)
