Acesse a linha de comandos shell Linux integrada do seu Android. Liberte seu geek interior!

Esta é uma nova versão do popular aplicativo "Android Terminal Emulator". O mesmo programa, apenas com novo nome.

Funções principais

+ Emulação completa do terminal Linux.
+ Múltiplas janelas.
+ Atalhos para lançadores.
+ Texto UTF-8. (Árabe, Chinês, Grego, Hebraico, Japonês, Coreano, Russo, Tailandês etc.)
+ Inteiramente gratuito. Sem anúncios, sem compras no aplicativo, sem ecrãs irritantes, nada.

FAQ rápido:

+ Se estiver a ter problemas ao digitar o texto, experimente instalar a aplicação gratuita "Hacker's Keyboard IME" (especialmente se tiver um aparelho Samsung ou HTC).
+ Esta aplicação não é um emulador de jogos.
+ Esta aplicação não ajuda a fazer root no telefone ou mudar o IMEI.
+ Tem de saber (ou estar disposto a aprender) como usar a linha de comandos do Linux para usar esta aplicação.
+ Talvez queira instalar um conjunto de utilitários de linha de comandos como o "Busy Box" ou "Debian chroot" para adicionar comandos além dos que vêm integrados no seu aparelho Android.
+ Se vir um erro -505 ao tentar instalar esta aplicação no Android 5.0, significa que uma outra aplicação usa as mesmas permissões. Terá de desinstalar a outra aplicação para poder instalar o Terminal Emulator for Android (dizem que o "jrummy's Toolbox Pro" causa este problema)
