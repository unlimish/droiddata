Αποκτήστε πρόσβαση στην ενσωματωμένη γραμμή εντολών του Android.

Νέα έκδοση της δημοφιλούς εφαρμογής «Android Terminal Emulator». Το ίδιο πρόγραμμα, με νέο όνομα.

Κύρια Χαρακτηριστικά

+ Πλήρης εξομοίωση τερματικού Linux. 
+ Πολλαπλά παράθυρα. 
+ Συντομεύσεις εκκίνησης. 
+ Κείμενο UTF-8. (Αραβικά, Κινέζικα, Ελληνικά, Εβραϊκά, Ιαπωνικά, Κορεάτικα, Ρώσικα, Ταϊλανδέζικα, κτλ.)
+ Εντελώς δωρεάν. Καθόλου διαφημίσεις, καθόλου αγορές εντός εφαρμογής, καθόλου ενοχλητικές οθόνες, τίποτα.

Συνήθεις ερωτήσεις:

+ Εάν αντιμετωπίζετε προβλήματα με την εισαγωγή κειμένου, δοκιμάστε να εγκαταστήσετε το δωρεάν "Hacker's Keyboard IME". (Ειδικά αν έχετε συσκευή Samsung ή HTC.)
+ Αυτή η εφαρμογή δεν είναι εξομοιωτής παιχνιδιών.
+ Αυτή η εφαρμογή δεν θα σας βοηθήσει να κάνετε root το τηλέφωνό σας ή να αλλάξετε το IMEI του τηλεφώνου σας.
+ Πρέπει να γνωρίζετε (ή να είστε πρόθυμοι να μάθετε) πώς να χρησιμοποιήσετε τη γραμμή εντολών Linux για να χρησιμοποιήσετε αυτήν την εφαρμογή.
+ Μπορεί να θέλετε να εγκαταστήσετε ένα σύνολο βοηθητικών προγραμμάτων της γραμμής εντολών όπως το "Busy Box" ή το "Debian chroot" για να προσθέσετε εντολές πέρα από τις εντολές που έχουν κατασκευαστεί στη συσκευή σας Android.
+ Εάν λάβετε ένα σφάλμα -505 κατά την προσπάθεια εγκατάστασης αυτής της εφαρμογής στο Android 5.0, αυτό σημαίνει ότι κάποια άλλη εφαρμογή χρησιμοποιεί τα ίδια δικαιώματα. Θα πρέπει να απεγκαταστήσετε την άλλη εφαρμογή για να εγκαταστήσετε το Terminal Emulator για Android. (Οι άνθρωποι μου λένε ότι το "jrummy's Toolbox Pro" προκαλεί αυτό το πρόβλημα.)
