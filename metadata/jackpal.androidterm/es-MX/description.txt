Accede a la línea de comandos de Linux incorporada en tu Android. Libera tu geek interior!

Esta es una nueva versión de la popular aplicación "Emulador de terminales Android". El mismo gran programa, sólo que con un nuevo nombre.

Características principales

+ Emulación completa de terminal de Linux. Múltiples ventanas. Atajos del lanzador. Texto UTF-8. (árabe, chino, griego, hebreo, japonés, coreano, ruso, tailandés, etc.) Completamente gratis. Sin anuncios, sin compras en la aplicación, sin pantallas de regaño, nada.

Preguntas frecuentes rápidas:

+ Si tienes problemas para introducir el texto, intenta instalar el "IME del teclado del hacker" gratuito. (Especialmente si tienes un dispositivo Samsung o HTC.)
+ Esta aplicación no es un emulador de juegos.\N+ Esta aplicación no te ayudará a hacer root en tu teléfono o a cambiar el IMEI de tu teléfono.\N+ Necesitas saber (o estar dispuesto a aprender) cómo usar la línea de comandos de Linux para usar esta aplicación.
+ Es posible que desee instalar un conjunto de utilidades de línea de comandos como "Busy Box" o "Debian chroot" para añadir comandos más allá de los comandos que vienen incorporados en su dispositivo Android.
+ Si obtiene un error -505 al tratar de instalar esta aplicación en Android 5.0, significa que alguna otra aplicación está usando los mismos permisos. Tendrás que desinstalar la otra aplicación para poder instalar el emulador de terminal para Android. (La gente me dice que "jrummy's Toolbox Pro" causa este problema).
