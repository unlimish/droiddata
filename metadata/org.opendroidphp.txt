Description:
Web server with PHP and mySQL support that runs directly on your mobile
device. It requires 50mb of internal storage for installation on unrooted
phones and has following features:

* PHP5
* MYSQL
* MYSQL Monitor
* Lighttpd (ssl enabled)
* Nginx (ssl enabled)
