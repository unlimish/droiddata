Summary:Automatically updates the EPO.DAT on the Fairphone
Description:Automatically updates the EPO.DAT file on the Fairphone 1 (and probably other
Mediatek devices, like the Alcatel Onetouch 997D). Unlike other apps, that
promise to fix the EPO issue, this app allows for manual and *automatic*
updates.
The current version is an alpha version and only for testing purposes!
Tested with:
- Fairphone FP1 with the 'Stock Android OS' and busybox
- Alcatel Onetouch 997D testing is currently in progress
Needs root access in order to write the new EPO.DAT to /data/misc.
