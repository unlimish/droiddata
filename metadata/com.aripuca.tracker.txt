still can't be viewed within the app because an API key for Google Maps is
missing. You can still use another app that supports OpenStreetMap and gpx files
to view it though, such as <a href="https://f-droid.org/packages/com.robert.maps">RMaps</a> or
<a href="https://f-droid.org/packages/nl.sogeti.android.gpstracker">Open GPS Tracker</a>.
