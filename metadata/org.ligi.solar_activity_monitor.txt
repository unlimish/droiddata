Summary:Android Widget to show Solar Activity
Description:
This is a Widget to display the latest 3 Hourly Planetary(estimated Ap) K
Indices from <a href="http://www.swpc.noaa.gov/ftpdir/lists/geomag/AK.txt">swpc.noaa.gov</a>
to e.g. help UAS Pilots to be aware of possible GPS-Problems.

Thanks to <a href="http://mikrokopter.de">mikrokopter.de</a> for the impulse,
<a href="http://www.swpc.noaa.gov">swpc.noaa.gov</a> for the Data and Clemens -Nebukad-
Wronski for the Graphics!
