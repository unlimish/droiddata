閱讀放喺Barcodes 入面嘅資料或者建立QR code喺唔同裝置之間分享地址，聯絡人或文字。

支援以下barcode嘅種類：

* UPC-A 同 UPC-E
* EAN-8 同埋 EAN-13
* Code 39; 93; 128
* ITF
* Codabar
* RSS-14 (各種)
* QR Code
* Data Matrix
* Aztec ('beta'質量)
* PDF 417 ('alpha'質量)

請參閱網站入面嘅疑難排解同權限嘅詳細解釋。呢個App容許你分享聯絡人、程式及書籖喺QR Code內。以上解釋咗點解會需要聯絡人嘅權限。請見「前往程序發行人員網站」如下（https://github.com/zxing/zxing/wiki/Frequently-Asked-Questions）。

如果你部裝置冇喺度繼續掃瞄，請先去到設定入面嘅裝置除錯，然後熄咗所有選項之後再逐個開返再決定邊個係需要。如果試完無幫助，試吓清除裝置快存或設定內Android既設置。

如果試過晒都唔得，你又相信喺個App既問題，可以用以下落嚟嘅資料Send個訊息畀作者。如果Send嘅信息無呢個資料就無辦法解決，會被選擇忽略。

-裝置種類
- 一個基本簡介講下有乜唔掂: 崩潰? 唔掃描? 睇起嚟奇怪?
- 認知你已經試過上面嘅步驟
- 裝置日誌會喺開始時捕取
