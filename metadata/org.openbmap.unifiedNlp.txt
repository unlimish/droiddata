Summary:UnifiedNlp location provider (Radiocells.org)
Description:
<a href="https://f-droid.org/packages/com.google.android.gms">UnifiedNlp (no GAPPS)</a> wifi/cell geolocation backend that uses
<a href="https://radiocells.org">Radiocells.org</a> database for geolocation.

Location can be retrieved either online or completely offline if a local
database is already present.

<a href="https://f-droid.org/packages/org.openbmap">RadioBeacon</a> can be used to gather wifi and cell information.
