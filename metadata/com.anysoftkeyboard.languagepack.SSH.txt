Summary:Language pack for ASK
Description:
A handy keyboard for terminal emulators, ssh clients (e.g. <a href="https://f-droid.org/packages/org.connectbot">ConnectBot</a>)
and any application that requires a full keyboard. The keyboard learns your
frequently used commands as you use it (you need to add your common commands for
auto suggestion: only tested on <a href="https://f-droid.org/packages/sk.vx.connectbot">VX ConnectBot</a>).

* Command suggestion (comes with basic dictionary)
* Arrow keys for command history
* CTRL key e.g. CTRL+E to jump to end of line
* Tab key for completion (depends on shell)
* Special characters as pop-ups

Install <a href="https://f-droid.org/packages/com.menny.android.anysoftkeyboard">AnySoftKeyboard</a> first, then select the desired
layout from AnySoftKeyboard's Settings->Keyboards menu.
